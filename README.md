# 📄 نموذج جاهز - مساحة عملك للمستندات

**نموذج جاهز** هو تطبيق ويب حديث مصمم لتبسيط عملية إنشاء وتحرير وإدارة المستندات القانونية والإدارية الشائعة في العالم العربي. تم بناء التطبيق باستخدام HTML, CSS, و JavaScript فقط، مع التركيز على تجربة المستخدم السلسة والمرونة العالية.

## 🚀 الميزات الرئيسية

- **🎨 واجهة عصرية وداعمة للوضع الليلي:** تصميم أنيق ومريح للعين، مع إمكانية التبديل بين الوضع الفاتح والداكن وحفظ تفضيل المستخدم.
- **📚 مكتبة نماذج شاملة:** مجموعة واسعة من القوالب الجاهزة للاستخدام، مثل عقود الإيجار، البيع، الزواج، إقرارات الاستلام، وغيرها.
- **💾 مساحة عمل شخصية "مستنداتي":** لا تفقد عملك بعد الآن! يمكنك حفظ مسودات مستنداتك في متصفحك والعودة لتعديلها أو حذفها في أي وقت.
- **📱 تصميم متجاوب (Mobile-First):** تجربة استخدام مثالية على جميع الأجهزة، مع واجهة خاصة بالهواتف تضع جميع الإجراءات الهامة في متناول يدك.
- **✏️ تخصيص كامل:** إمكانية إضافة "بنود إضافية" للعقود لتلبية احتياجاتك الخاصة.
- **📄 تصدير فوري:** حمّل مستنداتك النهائية بصيغة PDF بضغطة زر، أو قم بنسخ النص مباشرة لاستخدامه في أي مكان آخر.
- **✍️ تحويل الأرقام إلى نصوص:** يقوم التطبيق تلقائيًا بتحويل المبالغ المالية من أرقام إلى نصوص باللغة العربية في العقود.
- **🔍 بحث سريع:** يمكنك العثور على النموذج الذي تحتاجه بسهولة من خلال شريط البحث.
- **💨 أداء فائق:** تطبيق خفيف وسريع يعمل بالكامل في المتصفح دون الحاجة لأي تثبيت أو اتصال بالخادم.

## 🛠️ التقنيات المستخدمة

- **HTML5**
- **CSS3** (مع متغيرات CSS للثيمات)
- **JavaScript (ES6+)**
- **Bootstrap 5.3** (لتصميم الواجهة)
- **jsPDF & html2canvas** (لتوليد ملفات PDF)
